# NGJ26-NOTHING-PIB

[![Itch.io](https://img.shields.io/badge/itch.io-%23FF0B34.svg?logo=Itch.io&logoColor=white)](https://kreas.itch.io/nemotito) [![Godot Engine](https://img.shields.io/badge/Godot-%23FFFFFF.svg?logo=godot-engine)]([#](https://godotengine.org/))

This is a game submitted for Nordic Game Jam, 2026.

Theme: **Nothing**.

## Description

**Little Nemo** is asleep...

I wonder what kitties dream about?

### Objective
Make the digit in the upper left corner go from 0 to 10

### Controls
| Action | Input |
|--|--|
Move | Left click (hold) in desired direction
Jump |	Left click
Charged jump | Left click (hold), then release
